# dwm-statusbar

[![Build Status](https://travis-ci.com/kamil1b/dwm-statusbar.svg?branch=master)](https://travis-ci.com/kamil1b/dwm-statusbar)

## Requirement

- cmake
- compiler with c++17 support
- X11

## Build

```sh
mkdir build & cd build
cmake ..
make
```

## Run

```sh
bin/dwm-statusbar
```

## Modules

Time modules

- Time
- UpTime

Battery modules

- BatteryLevel
- BatteryStatus
