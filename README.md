## Requirement

- cmake
- compiler with c++17 support

## Build

```sh
mkdir build & cd build
cmake ..
make
```

## Run

```sh
bin/dwm-statusbar
```

## Modules

Time modules

  - Time
  - UpTime

Battery modules

  - BatteryLevel
  - BatteryStatus
